22    /**
23    * This class implements fast, thread-safe format of a double value
24    * with a given number of decimal digits.
25    * <p>
26    * The contract for the format methods is this one:
27    * if the source is greater than or equal to 1 (in absolute value),
28    * use the decimals parameter to define the number of decimal digits; else,
29    * use the precision parameter to define the number of decimal digits.
30    * <p>
31    * A few examples (consider decimals being 4 and precision being 8):
32    * <ul>
33    * <li>0.0 should be rendered as "0"
34    * <li>0.1 should be rendered as "0.1"
35    * <li>1234.1 should be rendered as "1234.1"
36    * <li>1234.1234567 should be rendered as "1234.1235" (note the trailing 5! Rounding!)
37    * <li>1234.00001 should be rendered as "1234"
38    * <li>0.00001 should be rendered as "0.00001" (here you see the effect of the "precision" parameter)
39    * <li>0.00000001 should be rendered as "0.00000001"
40    * <li>0.000000001 should be rendered as "0"

43    * Originally authored by Julien Aym&eacute;.
